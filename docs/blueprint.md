# **App Name**: MeetingFlow

## Core Features:

- Meeting Creation and Scheduling: Create meetings with specified agendas, invite participants, and schedule times.
- Real-time Conversational Note-Taking: Record meeting notes in real-time through voice input and transcribe them to text.
- Intelligent Action Item Assignment: Assign action items to participants with deadlines, leveraging AI tool to ensure relevant information and clarity.
- Meeting History and Document Retrieval: Access and review past meeting details, notes, and assigned action items.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) for a calm and professional feel.
- Background color: Light gray (#ECEFF1) for a clean and neutral base.
- Accent color: Deeper blue (#1E88E5) for key actions and highlights, creating contrast without overwhelming.
- Body and headline font: 'Inter' for a modern and readable interface.
- Use minimalist line icons in blue to represent meeting actions and elements.
- Maintain a clean and structured layout, using cards and dividers to organize content effectively.
- Subtle transitions and animations for a smooth and engaging user experience when navigating through meeting details and notes.
